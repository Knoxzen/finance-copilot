# Finance Copilot

AI-powered personal finance platform designed to help users understand spending habits, track financial goals, and receive intelligent financial guidance through modern backend engineering and GenAI.

---

## Vision

Finance Copilot is a SaaS-style personal finance platform that combines:

* Financial analytics
* Expense categorization
* Goal planning
* AI-powered financial insights

The project is being built as a production-grade engineering portfolio focused on:

* Java
* Spring Boot
* Microservices
* Event-Driven Architecture
* GenAI
* Cloud Deployment
